# 数组 
>  数组与标准库 类型 vector 类似，都存放类型相同对象的容器。
>  需要通过其所在的位置访问对象。
>  与vector不同的是，数组大小确定不变，不能随意向数组中增加元素, 数组不允许拷贝，vector允许拷贝。
